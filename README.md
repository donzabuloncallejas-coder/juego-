# Cuidemonos en Internet - Escape Room Educativo

Juego 2D animado para ninas, ninos y adolescentes sobre seguridad digital.

## Como ejecutar

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar en desarrollo:

```bash
npm run dev
```

3. Abrir en navegador:

```text
http://localhost:3000
```

## Salas del juego

- Ciberacoso
- Fake News
- Suplantacion de identidad
- Grooming

Sin registro y sin panel administrativo.
